Recursos adicionales

Este manual sólo ofrece una visión superficial de lo que puede hacer con Voyant Tools y los tipos de análisis de texto que puede realizar. Para obtener más información sobre Voyant Tools, consulta los siguientes recursos:

  • Voyant Help Guide
  • Voyant Tools GitHub

Para poder seguir los pasos de este manual es necesario descargar los siguientes ficheros zip:

  • Corpus de novelas del siglo XIX
  • Fuero Real (BUR, FRL, FRS)
  • Fuero Real (FRS: capítulos)

Los diferentes corpus están también disponibles en línea en el servidor de Voyant Tools:

  • Novelas siglo XIX: https://voyant-tools.org/?corpus=fb8fe0f6379a5cde970d67df40886f5c
  • Fuero Real (BUR, FRL, FRS): https://voyant-tools.org/?corpus=28c28166b1f3f68a4e5eca2e6b9bcf61
  • Fuero Real (FRS: capítulos): https://voyant-tools.org/?corpus=0dfea482c3a5c3b9ae1ca55f2ba7527b
